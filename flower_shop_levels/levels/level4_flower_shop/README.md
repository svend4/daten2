# Уровень 4 — Node.js + Express + EJS + SQLite

Структура (как в чате 14.12.2025):

```
flower_shop/
├── server.js              # Backend
├── package.json           # Зависимости npm
├── flowers.db             # База данных (SQLite, уже заполнена)
├── views/                 # EJS шаблоны
│   ├── index.ejs
│   ├── product.ejs
│   └── order.ejs
└── public/
    └── style.css
```

## Запуск
```bash
npm install
npm start
```
Открыть: http://localhost:3000
